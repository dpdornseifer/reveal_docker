What is Containerization? / What is Docker?

Container vs. Virtualization

  • Faster Spinn up
  • Small footprint

But how does it work?

  • Dates back to year 2000
  • FreeBSD jails
  • Solaris Zones
  • OpenVZ
  • LXC (Linux Containers)

How is it used in the Cloud and why is it so popular

  • Immutable infrastructure (source)
    a.k.a immutable servers, phoenix servers to avoid drift - differences between servers when they are supposed to be identical containers can share directories, network stack ...
  • Less overhead
  • Four to six times the number of server application instances compared to Xen, KVM (http://www.zdnet.com/article/what-is-docker-and-why-is-it-so-darn-popular/)
  • Faster ...
  • Flexibility to spin up whole integration testing environment

The Docker Ecosystem

How can I use it?

Dockerfile

							
FROM python:3.5

# copy SAP specific certs to make the SSL validation work
ADD certificates /usr/local/share/ca-certificates/
RUN update-ca-certificates

# create directory for app
RUN mkdir /app

# install required python 3.5 libs
COPY requirements.txt /app/
RUN pip3 --proxy proxy.wdf.sap.corp:8080 install -r /app/requirements.txt

# add handler for routes
ADD handler /app/handler

CMD python /app/app.py
EXPOSE 8080
